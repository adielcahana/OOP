1) כאשר מתקבל ביטוי של חזקה ביטוי בחזקת לוגריתם שהבסיס שלו הוא חזקה הפישוט הוא הלוגרתים. לדוגמא: x^log(x,5)=>5

2) ביטוי שמעלים בחזקת ביטוי שגם הוא מועלה בחזקה הפישוט שלו הוא הביטוי הראשון בחזקת הביטוי השני כפול השלישי. לדוגמא:
(x^y)^z => x^(y*z)

3) זהות טריגונומטרית: cos(90 + x) => - sin(x)

4) זהות טריגונומטרית: sin(90 + x) => cos(x)

5) צמצום של המינוסים כאשר יש יותר ממינוס אחד כל שני מינוסים רצופים מבטלים זה את זה (שלילה כפולה) לדוגמא:
-(-(-(x +y) => -(x +y)

6) ביטוי בחזקת 0 = 1 לדוגמא: x^0 => 1

7) ביטוי בחזקת 1 = ביטוי עצמו לדוגמא: x^1 => x

8) חילוק של ביטוי ב1 = ביטוי עצמו לדוגמא: (x +y)/1 => x+y
